# 🎮 Концепция: гремлин-компаньон на ленте X

Синтез дизайн-панели (4 линзы + ресёрч Neko/Shimeji/eSheep/Clippy/BonziBuddy/
Desktop Goose/Tamagotchi). Это карта продукта; что уже сделано — помечено ✅.

## Core loop

Гремлин живёт поверх твоей ленты (оверлей, реальный DOM не трогается). В простое
занимается своими делами (зевает, крадётся за курсором, дремлет в углу) — это и есть
«жизнь». Скучая или голодая, **эскалирует в визуальную пакость-сюрприз** (трещина,
кража буквы, угон иконки) — variable reward, без метронома. Ты реагируешь одним из
трёх жестов приручения:

- **Погладить** — зажать и водить курсором по телу (мурлычет, сердечки).
- **Прибрать** — клик/протереть по самой пакости (зверь собирает беспорядок обратно).
- **Покормить = лайк** — твой осознанный клик по канючащему зверю у кнопки ❤ ставит
  ОДИН настоящий лайк (паттерн прототипа). ✅

Каждая реакция гасит шкоду, чуть растит **связь (bond)** и насыщает зверя. Чем выше
bond — тем зверь ручнее: меньше пакостит, эффектнее «приглашает» к лайку, встречает
при заходе. Держится на эффекте Зейгарник (открытая шкала шкоды/сытости) и baby-schema
(большие глаза), но без жёсткой смерти и вины 24/7 — ставки мягкие.

## Настроение и связь (3 скрытых параметра в storage, тикают по реальному времени)

1. **Сытость / Дикость** — ось «дикости»: падает со временем, растёт от кормёжки;
   высокая = чаще и злее пакости, низкая = смирный зверь.
2. **Mood / Energy** — мимика и темп (глаза, палитра тела через CSS-переменные,
   множитель скорости); на нуле зверь засыпает в углу.
3. **Bond** — копится медленно (глажка/кормёжка/стрик), падает от запущенности.
   Лестница доверия: дикий → ручной → преданный → компаньон.

Эскалация **никогда не вредит данным X** — игнор лишь визуально нарастает (под
cooldown + лимитом эффектов). Любая реакция мгновенно и обратимо всё гасит.

## v1 — минимальный играбельный набор

| Механика | Статус | Безопасность |
|---|---|---|
| **Кормёжка = лайк** (клик по зверю → 1 настоящий лайк, привязка к ID, подтверждение) | ✅ есть | human-initiated |
| **Шкода + успокоение**: трещина и кража буквы | ✅ есть (без шкалы дикости) | visual-only |
| **Поглаживание-приручение** (возишь курсором → мурлык, +bond) | ⬜ | visual-only |
| **Drag-and-throw** (хватать/швырять зверя, пружинная инерция) | ⬜ | visual-only (лайк выключен во время drag) |
| **Idle-ритуалы** (зевок, сон, умывание, крадётся за мышью, прячется) | 🟡 частично (walk/beg/happy) | passive |
| **Вкл/тихий режим + ползунок вредности** | 🟡 есть вкл/выкл и выбор героя | passive |

Следующий шаг для «живости»: связать пакости со **шкалой дикости** (голодный/скучающий
→ чаще пакостит, сытый/глаженый → смирный) и добавить **поглаживание**.

## Бэклог (по убыванию ценности)

- Лестница доверия (4 уровня: дистанция, частота канюченья, поведение при заходе).
- Ежедневное приветствие + стрик (якорь привычки вернуться).
- Эволюция и скины по порогам bond (моддинг = вирусность).
- Последствия запущенности и побег (мягко, обратимо).
- «Сегодняшние шалости» — мини-достижения с панчлайнами (вайб Untitled Goose Game).
- Mood-sniff: зверь «нюхает» пост под собой, меняет глаза/пузырь.
- Новые пакости: угон иконки, каляки на аватарках, подмена цифр статистики.
- Трюки/дрессировка (двойной клик = сальто, удержание = «сидеть»).
- **Twitter-нативные обёртки (тот же human-initiated паттерн):** Crack→открыть reply
  (текст и отправку пишешь сам), угон автора→follow (только видимый follow), и т.п.
- Настоящий спрайт-лист (walk/jump/hit/happy/sleep) вместо CSS-трансформов.
- Дневной лимит лайков + онбординг-дисклеймер (из ревью, к публичному релизу).
- Звуковой движок с мьютом (писк/мурлык/звон стекла).

## Анти-Clippy (правила, чтобы не бесил)

- Замирает при наборе текста и открытой модалке (✅ в коде) — главный урок против Clippy.
- Никакого угадывания намерений («похоже, вы пишете…»).
- Пакости — только сюрприз под cooldown + лимит одновременных; есть ползунок вредности.
- Вариативность и затухание приевшихся шуток (анти-повтор).
- Зверь НИКОГДА не делает реальное действие в X сам — только по клику по нему;
  во время drag реальные действия выключены (исключаем случайный лайк).
- Все эффекты визуальные, поверх, мгновенно обратимы; клики по ленте не перехватываются
  (пакость, блокирующая реальный клик = баг).
- Мгновенная пауза/мьют/прятки, честный off; ничего не переоткрывается само.
- Перемежать хаос wholesome-моментами (благодарность, сердечки, приветствие).
- Без жёсткой смерти и вины 24/7; запущенность обратима.
